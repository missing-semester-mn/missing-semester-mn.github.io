Үйлдлийн системийнхээ тохиргооны цэснээс илүү өргөн хүрээнд системийг өөрийн
хүслээр өөрчлөн тохируулах олон боломж байдаг.

# Гарын товчлуурын дахин хуваарилалт (Keyboard remapping)

Таны гар дээр магадгүй таны байнга ашигладаггүй товчлуурууд байдаг байх. Тэдгээр
ашиглагддаггүй товчлууруудыг хэрэгцээгүй байлгахын оронд дахин хуваарилж (remap)
хэрэгтэй зүйл хийдэг болгон өөрчилж болно.

## Бусад товчлуур руу чиглүүлэх

Хамгийн энгийн арга бол товчлуурыг өөр нэг товчлуурын үүрэгтэй ижил болгож
хуваарилах юм. Жишээлбэл, хэрэв та Caps Lock товчлуурыг тийм ч их ашигладаггүй
бол түүнийг илүү хэрэгцээтэй товчлуурын үүрэгтэй болгон өөрчилж болно. Хэрэв та
Vim ашигладаг бол Caps Lock-ийг Escape товчлуур болгон өөрчлөх нь түгээмэл
сонголт байдаг.

macOS дээр та зарим энгийн дахин хуваарилалтыг Системийн Тохиргооны Гарын
тохиргоо (Keyboard settings in System Preferences) хэсгээс хийж болох бөгөөд
илүү нарийн тохируулга хийхийн тулд тусгай программ хангамж ашиглах шаардлагатай
болно.

## Дурын тушаалд холбох

Товчлууруудыг зөвхөн өөр товчлуур болгох албагүй: тодорхой товчлуур (эсвэл
товчлууруудын хослол)-ыг дурын тушаал ажиллуулдаг болгож тохируулж болно.
Жишээлбэл, Command+Shift+T дарахад шинэ терминалын цонх нээгддэг болгож болно.

# ҮС-ийн далд тохиргоог өөрчлөх

## macOS

macOS нь `defaults` тушаалаар дамжуулан маш олон хэрэгцээтэй далд тохиргоог ил
гаргадаг. Жишээлбэл, далдлагдсан аппликейшнуудын Dock дээрх дүрсийг хагас
тунгалаг болгохын тулд:

```shell
defaults write com.apple.dock showhidden -bool true
```

Боломжтой бүх тохиргооны нэгдсэн жагсаалт гэж байдаггүй ч та онлайнаар Mathias
Bynens-ийн [.macos](https://github.com/mathiasbynens/dotfiles/blob/master/.macos)
гэх мэт тусгай жагсаалтуудыг олж харах боломжтой.

# Цонхны удирдлага (Window management)

## Хавтанцар цонхны удирдлага (Tiling window management)

[Хавтанцар цонхны удирдлага](https://en.wikipedia.org/wiki/Tiling_window_manager)
(tiling window manager) нь цонхнуудыг хооронд нь давхцуулахгүйгээр дэлгэцэнд
хуваан байршуулж удирддаг арга юм. Хэрэв та Unix-д суурилсан үйлдлийн систем
ашигладаг бол хавтанцар цонхны менежерийг суулгаж болно; Windows эсвэл macOS
ашигладаг бол үүнтэй төстэй ажиллагааг үзүүлдэг программуудыг суулгах боломжтой.

## Дэлгэц хоорондын удирдлага (Screen management)

Та цонхнуудыг олон дэлгэц хооронд шилжүүлж удирдахад туслах гарын товчлолуудыг
тохируулж болно.

## Байршил (Layouts)

Дэлгэц дээр цонхнуудыг өөрийн гэсэн тодорхой байршлаар гараар байршуулж цаг
үрэхийн оронд түүнийг скрипт бичин автоматжуулж, байршлыг маш хурдан
идэвхжүүлдэг болгож болно.

# Дасгалууд

1. Өөрийн Caps Lock товчлуурыг Escape, Ctrl эсвэл Backspace гэх мэт байнга
   ашигладаг өөр зүйлд хэрхэн шилжүүлж тохируулахыг олж мэдээрэй.

1. Шинэ терминалын цонх эсвэл шинэ вэб хөтчийн цонхыг нээх тусгай гарын
   товчлолыг систем даяар ажилладаг болгон үүсгээрэй.
